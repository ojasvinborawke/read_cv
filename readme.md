# CSV Processing with CompletableFuture (Java)

## Overview
This project demonstrates:

- Reading CSV files using OpenCSV
- Converting CSV rows into Java objects (`Product`)
- Concurrent file loading using `CompletableFuture`
- Merging datasets using Cartesian product logic
- Stream API operations (grouping, sorting, aggregation)

---

## Tech Stack
- Java 17+
- OpenCSV
- Java Streams API
- CompletableFuture (async processing)
